Map Rotater

Angle

Choose a map .cl file

Instructions:

  1. In CM, export a map content item by going to the Content menu, and hitting Export Library. Only export one map item at a time
  2. Upload that .cl item to this site and choose 90, 180, or 270 degrees.
  3. You will recieve a file that contains the rotated data. Import it into CM, it will have a new name, so you don't have to worry about a conflict.
  4. Replace the floor path images with the rotated paths.

Notes:

This tool has been tested with map cl files from 4.3.2, 4.5, 4.6.2, and 5.0. I've looked at the xml for others and it should be fine. If you find a bug, e-mail me with the cl file. This tool calculates angles in a counter clockwise manner.

Limitations:

  • Floor Images must be locally pathed (not served from a web server), this is becasue images that are served via http do not get copied into a cl item, and I need them to determine the canvas size.
  • This tool makes the assumption that your rotated map is the same scale as the original. Also aspect ratios are inverted. If your original was 16:9 the rotated will be 9:16.
  • This assumes that map assets on the map haven't been moved in a rotated state, however this can't always be gaurenteed as labels can get in the way. You may have to move a couple points to match the rotated image.
  • Perspective maps are not supported.
